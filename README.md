# Login_and_Registration
A simple Login and Registration UI for android application

It consits of following tasks:
* Multiple Activities
* UI Elements
  * Text View
  * Toast Notifications
  * Buttons
  * Dialogs
* Fragments
* Action Bars
* Imolicts and Explicits Intents
* SQLite Database
